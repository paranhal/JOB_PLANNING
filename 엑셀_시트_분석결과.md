# 엑셀 시트 분석 결과

**상태:** 기획서 기준 재구축. 본 문서는 **참고·이관용**으로만 사용한다.

**설계 기준:** 기획서(`기획_업무일지_프로그램_요구사항.md`)에 명시된 대로, **통합 구조**가 기준이며 시트 이름·개수에 맞출 필요 없음. 기존 엑셀은 **데이터 이관(가져오기)·내보내기** 시 매핑 참고용으로만 활용한다.

- 상세 시트 목록·컬럼은 기획서 **부록** 및 필요 시 본 문서에 재정리한다.
- 재구축 시 기획서 §0-4를 우선하고, 이관/내보내기 설계 시 본 문서를 참고한다.
