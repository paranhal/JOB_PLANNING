## 1. 정기점검 유지보수계획표(기획서 §17.11)

시스템에서 생성하는 **xlsx**는 월별 시트 `1월`~`12월`을 포함하며, 각 시트는 **일~토 7열 달력** 형태이다. 데이터는 `maintenance_plans` + `maintenance_visits`(+ 표시명은 `maintenance_site_config.short_name` 또는 `customers.org_name`)에서 읽는다.

| 출력 항목 | 데이터 출처 | 비고 |
|-----------|---------------|------|
| 월 제목 (예: 2026년 3월) | `maintenance_plans.plan_year` + 시트 월 | 녹색 강조 스타일 |
| 일자 칸 상단 숫자 | `visit_date`의 일 | |
| 사이트명 줄 목록 | `maintenance_visits` 당일 다건 + `short_name` | 자동 줄바꿈 |
| 글자색 (녹/적/청) | `maintenance_visits.entry_category` | `normal` / `fixed` / `office` |
| 토·일 칸 배경 | (달력 위치) 주말 열 | 공휴일은 방문이 없으면 빈 칸 |

**파일명 규칙(권장):** `YYYYMMDD유지보수계획.xlsx` — 구현은 다운로드 시점 기준 날짜를 사용한다.

**가져오기(역방향):** 후속 단계에서 동일 레이아웃 xlsx를 읽어 `maintenance_visits`를 갱신할 경우, 시트명·날짜 셀 파싱 규칙을 본 절과 맞춘다.
