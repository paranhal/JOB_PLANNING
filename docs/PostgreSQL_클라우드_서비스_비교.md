# PostgreSQL 클라우드 서비스 비교 — 국내·저가·1년 사용 관점

국내 서비스 위주로, **저렴한 가격으로 1년 정도** 사용할 수 있는 PostgreSQL 클라우드 옵션을 정리했습니다. (검색 기준: 2025년 상반기)

---

## 1. 국내(한국) 서비스

### 1-1. 카카오클라우드 (Kakao Cloud)

| 항목 | 내용 |
|------|------|
| **제품** | PostgreSQL (관리형) |
| **최소 사양** | m2a.large (2vCPU, 8GB RAM) |
| **가격** | **시간당 248원** → **월 약 178,560원** (24시간×30일 기준) |
| **1년 예상** | 약 **214만 원** 수준 |
| **추가** | 스토리지 0.16원/GiB/시간, 백업 0.14원/GiB/시간 |
| **링크** | https://kakaocloud.com/services/postgresql/pricing |

- 국내 서비스 중 **가격이 공개된** 대표 옵션.
- 더 작은 인스턴스가 필요하면 문의로 “소규모 개발용” 요금 확인 가능.

---

### 1-2. 네이버 클라우드 플랫폼 (NCP)

| 항목 | 내용 |
|------|------|
| **제품** | Cloud DB for PostgreSQL |
| **요금** | **종량제** (사용 시간·사양별) |
| **확인 방법** | 콘솔 > 요금 > 상품별 요금안내, 또는 **요금 계산기** 사용 |
| **링크** | https://www.ncloud.com/product/database/cloudDbPostgresql, 요금 계산기: https://www.ncloud.com/v2/charge/calc |

- 최소 사양·스토리지 선택 후 계산기에서 **월/년 예상 비용** 확인 가능.
- 카카오보다 작은 사양이 있으면 1년 비용을 더 낮출 수 있음.

---

### 1-3. NHN Cloud

| 항목 | 내용 |
|------|------|
| **제품** | RDS for PostgreSQL |
| **요금** | 인스턴스 타입(m2/c2/r2/x1)·스토리지(20GB~)·리전별 상이 |
| **확인 방법** | **요금 계산기**에서 DB 선택 후 PostgreSQL·리전·사양 입력 |
| **링크** | https://www.nhncloud.com/kr/pricing/calculator (Database > RDS for PostgreSQL) |

- 리전: 한국(평촌/판교), 일본, 미국 등. 한국 선택 시 국내 리전 사용 가능.
- “저렴하게 1년” 쓰려면 계산기에서 **최소 인스턴스 + 최소 스토리지**로 월 비용 산출 후 ×12로 확인.

## 2. 1년 저비용·무료에 가까운 옵션

“국내만”이 아니어도 된다면, **1년 동안 저렴/무료**로 쓰기 좋은 선택지입니다.

### 2-1. AWS RDS for PostgreSQL (프리 티어)

| 항목 | 내용 |
|------|------|
| **기간** | **가입 후 12개월** (프리 티어) |
| **제공** | 매월 **750시간** (소형 인스턴스 1대 24시간 구동 가능 수준) |
| **사양** | db.t4g.micro 등 소형 인스턴스, 스토리지 20GB |
| **1년 비용** | 프리 티어 조건 내 사용 시 **0원** (12개월 후부터 종량 과금) |
| **참고** | 리전을 **아파넥서울(ap-northeast-2)** 로 두면 한국 리전 사용 |
| **링크** | https://aws.amazon.com/ko/rds/free/ |

- **국내 접속 가능 + 1년 무료**로 쓰고 싶을 때 가장 흔히 추천되는 옵션.

---

### 2-2. Supabase (무료 플랜)

| 항목 | 내용 |
|------|------|
| **제공** | PostgreSQL 호스팅 + API·인증 등 |
| **무료 플랜** | 프로젝트 2개, DB **500MB/프로젝트**, 5GB 이그레스 등 |
| **1년 비용** | 무료 플랜만 쓸 경우 **$0** (제한 내 사용 시) |
| **참고** | 서버가 해외 리전이므로 국내에서 접속 지연 가능. 개발·소규모 서비스에 적합 |
| **링크** | https://supabase.com (요금: 플랫폼 내 Billing) |

---

### 2-3. Neon (무료 플랜)

| 항목 | 내용 |
|------|------|
| **제공** | Serverless PostgreSQL |

| **1년 비용** | 무료 플랜만 사용 시 **$0** |
| **참고** | 해외 리전. 소규모·개발·테스트용으로 적합 |
| **링크** | https://neon.tech |

## 3. “국내 + 저렴 + 1년” 요약

| 목적 | 추천 |
|------|------|
| **반드시 국내 업체만** | **카카오클라우드** (월 약 17만 원대부터, 1년 약 214만 원). 더 저렴한 건 **네이버/NHN 요금 계산기**로 최소 사양 조합 확인. |
| **1년 가능한 한 저비용** | **AWS RDS 프리 티어**(12개월 0원, 서울 리전 선택). 12개월 이후는 소형 인스턴스 종량제. |
| **완전 무료에 가깝게 1년** | **Supabase** 또는 **Neon** 무료 플랜 (해외 리전, 개발/소규모용). |

---

## 4. 선택 시 참고

- **데이터 국내 보관 의무**가 있으면: 국내 리전 제공 업체(카카오, 네이버, NHN, AWS 서울 등) 중에서 선택.
- **비용만 최소화**하면: AWS 프리 티어 1년 → 이후 소형 인스턴스 또는 Supabase/Neon 무료 플랜 검토.
- **정확한 1년 비용**이 필요하면:  
  - 국내: 네이버/NHN **요금 계산기**에서 최소 PostgreSQL 구성을 넣어 보고, 월 비용 × 12.  
  - AWS: RDS 요금표에서 db.t4g.micro + 20GB 스토리지로 12개월 이후 예상 비용 확인.

위 내용은 각 사이트 요금·정책이 바뀔 수 있으므로, **실제 가입·결제 전에 해당 서비스 공식 요금 페이지에서 한 번 더 확인**하는 것이 좋습니다.
